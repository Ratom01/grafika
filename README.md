# Grafika
Grafika kurzusra készített projekt. Javascriptben megírt modelezés threejs és blender használatával.

A kurzus folyamán egy modelezni kellett mozgó, változó, irányítható objektumokat.
- A szobában folyamatosan mozog egy játékbeli karakter.
- Eltávolítható falak a szoba átlátásához.
- Blenderben modelezett karakter, illetve szekrény.
- Mozgó nap, ami átállítható hold textúrára.
- Irányítható labda a szobában.

## Megnyitása:
Mivel hálózati kéréseket indít a projekt, ezért szükség van szervere, ezért ajánlott IDE használata. A projekt Pycharm-ban készült.
